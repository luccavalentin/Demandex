# Configuração do Git

Para salvar as alterações no Git, execute os seguintes comandos no terminal, dentro do diretório do projeto:

```bash
git init
git add components/ app/globals.css .gitignore
git commit -m "feat: Melhorias no sidebar - menu colapsável, cores dos ícones, layout responsivo e mobile otimizado"
```

Ou execute o arquivo `init-git.bat` que foi criado na raiz do projeto.
